# My Project

This project was created using [Better-T Stack](https://github.com/AmanVarshney01/Better-T-Stack).

## Getting Started

1. Install dependencies:
```bash
bun install
```

2. Set up database:
```bash
bun run db:local
bun run db:push
```

3. Start development:
```bash
bun run dev
```

Open [http://localhost:3001](http://localhost:3001) to view your app.
